##  RUN ollama
```
ollama run mistral
```
##  EXIT ollama
```
\bye
```
##  STOP ollama running in background
```
ollama stop mistral
```
##  LIST all ollama models in your machine
```
ollama list
```
##  DELETE ollama model
```
ollama rm mistral
```
##  SHOW all ollama running model
```
ollama ps
```
##  API to run
```
ollama serve
```
##  CREATE new model from exiting
```
ollama create harris -f ./Modelfile
```
